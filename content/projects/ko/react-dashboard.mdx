# React 대시보드 애플리케이션

실시간 데이터 시각화와 사용자 관리 기능을 제공하는 현대적인 대시보드 애플리케이션입니다.

## 프로젝트 개요

이 프로젝트는 기업의 다양한 데이터를 실시간으로 모니터링하고 시각화할 수 있는 웹 기반 대시보드입니다. React와 Chart.js를 활용하여 직관적이고 반응형인 사용자 인터페이스를 제공합니다.

## 주요 기능

### 📊 데이터 시각화
- **실시간 차트**: Chart.js를 사용한 다양한 차트 타입 지원
- **대시보드 위젯**: 사용자 정의 가능한 위젯 시스템
- **데이터 필터링**: 날짜, 카테고리별 데이터 필터링
- **내보내기 기능**: PDF, Excel 형식으로 데이터 내보내기

### 👥 사용자 관리
- **역할 기반 접근 제어**: 관리자, 사용자, 게스트 권한 관리
- **사용자 프로필**: 개인화된 대시보드 설정
- **알림 시스템**: 실시간 알림 및 이메일 알림

### 🔧 관리 기능
- **데이터 소스 관리**: 다양한 API와 데이터베이스 연결
- **위젯 커스터마이징**: 드래그 앤 드롭으로 대시보드 레이아웃 변경
- **설정 관리**: 시스템 설정 및 사용자 환경 설정

## 주요 컴포넌트

### 대시보드 위젯

```javascript
// DashboardWidget.jsx
import React from 'react';
import { Card, CardContent, Typography } from '@mui/material';
import { Line } from 'react-chartjs-2';

const DashboardWidget = ({ title, data, type = 'line' }) => {
  const chartOptions = {
    responsive: true,
    maintainAspectRatio: false,
    plugins: {
      legend: {
        position: 'top',
      },
    },
  };

  return (
    <Card sx={{ height: '100%' }}>
      <CardContent>
        <Typography variant="h6" gutterBottom>
          {title}
        </Typography>
        <div style={{ height: '300px' }}>
          {type === 'line' && <Line data={data} options={chartOptions} />}
          {type === 'bar' && <Bar data={data} options={chartOptions} />}
          {type === 'pie' && <Pie data={data} options={chartOptions} />}
        </div>
      </CardContent>
    </Card>
  );
};

export default DashboardWidget;
```

### 실시간 데이터 훅

```javascript
// useRealtimeData.js
import { useState, useEffect } from 'react';
import io from 'socket.io-client';

const useRealtimeData = (endpoint) => {
  const [data, setData] = useState(null);
  const [loading, setLoading] = useState(true);
  const [error, setError] = useState(null);

  useEffect(() => {
    const socket = io(process.env.REACT_APP_API_URL);
    
    socket.on('connect', () => {
      console.log('Connected to server');
      setLoading(false);
    });

    socket.on(endpoint, (newData) => {
      setData(newData);
    });

    socket.on('error', (err) => {
      setError(err);
      setLoading(false);
    });

    return () => {
      socket.disconnect();
    };
  }, [endpoint]);

  return { data, loading, error };
};

export default useRealtimeData;
```

## 보안 고려사항

### 1. 인증 및 권한
```javascript
// JWT 토큰 검증 미들웨어
const authenticateToken = (req, res, next) => {
  const authHeader = req.headers['authorization'];
  const token = authHeader && authHeader.split(' ')[1];

  if (!token) {
    return res.sendStatus(401);
  }

  jwt.verify(token, process.env.ACCESS_TOKEN_SECRET, (err, user) => {
    if (err) return res.sendStatus(403);
    req.user = user;
    next();
  });
};
```

### 2. 데이터 검증
```javascript
// 입력 데이터 검증
const validateData = (data) => {
  const schema = Joi.object({
    name: Joi.string().required(),
    value: Joi.number().min(0).required(),
    date: Joi.date().required()
  });

  return schema.validate(data);
};
```

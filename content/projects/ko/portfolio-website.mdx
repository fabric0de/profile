# 개인 포트폴리오 웹사이트

현대적이고 반응형인 개인 포트폴리오 웹사이트를 Next.js 14와 Tailwind CSS를 사용하여 구축했습니다.

## 프로젝트 개요

이 프로젝트는 개발자로서의 경험과 프로젝트를 효과적으로 보여주기 위한 개인 포트폴리오 웹사이트입니다. 최신 웹 기술을 활용하여 빠르고 사용자 친화적인 경험을 제공합니다.

## 주요 기능

### 🚀 성능 최적화
- **Next.js 14 App Router**: 최신 Next.js 기능을 활용한 서버 사이드 렌더링
- **이미지 최적화**: Next.js Image 컴포넌트를 사용한 자동 이미지 최적화
- **코드 스플리팅**: 페이지별 자동 코드 분할로 초기 로딩 시간 단축

### 🌐 다국어 지원
- **i18n 국제화**: 한국어, 영어, 일본어 지원
- **동적 라우팅**: URL 기반 언어 전환
- **SEO 최적화**: 각 언어별 메타데이터 및 사이트맵 생성

### 📱 반응형 디자인
- **모바일 퍼스트**: 모든 기기에서 완벽한 사용자 경험
- **Tailwind CSS**: 유틸리티 퍼스트 CSS로 일관된 디자인 시스템
- **다크 모드**: 시스템 설정 기반 자동 다크 모드 지원

### 📝 콘텐츠 관리
- **MDX 지원**: 마크다운과 JSX를 결합한 풍부한 콘텐츠 작성
- **Contentlayer**: 타입 안전한 콘텐츠 관리
- **자동 생성**: 블로그 포스트와 프로젝트 자동 인덱싱

## 기술 스택

### Frontend
- **Next.js 14**: React 기반 풀스택 프레임워크
- **React 18**: 최신 React 기능 활용
- **TypeScript**: 타입 안전성 보장
- **Tailwind CSS**: 유틸리티 퍼스트 CSS 프레임워크

### 콘텐츠 관리
- **MDX**: 마크다운과 JSX 결합
- **Contentlayer**: 정적 콘텐츠를 위한 타입 생성기
- **rehype-pretty-code**: 코드 블록 문법 하이라이팅

### 배포 및 호스팅
- **Vercel**: Next.js 최적화 호스팅 플랫폼
- **GitHub Actions**: 자동 배포 파이프라인
- **도메인**: 커스텀 도메인 연결

## 프로젝트 구조

```
├── app/                    # Next.js App Router
│   ├── [locale]/          # 다국어 라우팅
│   │   ├── blog/          # 블로그 페이지
│   │   ├── projects/      # 프로젝트 페이지
│   │   └── about/         # 소개 페이지
│   ├── globals.css        # 전역 스타일
│   └── layout.tsx         # 루트 레이아웃
├── components/            # 재사용 가능한 컴포넌트
│   ├── Layout.tsx         # 메인 레이아웃
│   ├── PostCard.tsx       # 블로그 포스트 카드
│   └── ProjectCard.tsx    # 프로젝트 카드
├── content/               # MDX 콘텐츠
│   ├── blog/              # 블로그 포스트
│   └── projects/          # 프로젝트 설명
└── lib/                   # 유틸리티 함수
    ├── content.ts         # 콘텐츠 관리 함수
    └── types.ts           # TypeScript 타입 정의
```

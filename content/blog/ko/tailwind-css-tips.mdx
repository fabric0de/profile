# Tailwind CSS 실전 팁과 트릭

Tailwind CSS는 빠른 프로토타이핑과 일관된 디자인 시스템을 제공하는 유틸리티 퍼스트 CSS 프레임워크입니다. 이 글에서는 Tailwind CSS를 더 효율적으로 사용하는 방법을 알아보겠습니다.

## 1. 커스텀 유틸리티 클래스 만들기

### @apply 디렉티브 사용

```css
/* styles.css */
@tailwind base;
@tailwind components;
@tailwind utilities;

@layer components {
  .btn-primary {
    @apply bg-blue-600 text-white px-4 py-2 rounded-lg hover:bg-blue-700 transition-colors;
  }
  
  .card {
    @apply bg-white rounded-lg shadow-md p-6 border border-gray-200;
  }
}
```

### 컴포넌트 클래스 활용

```javascript
// Button.jsx
const buttonVariants = {
  primary: 'bg-blue-600 text-white hover:bg-blue-700',
  secondary: 'bg-gray-200 text-gray-900 hover:bg-gray-300',
  danger: 'bg-red-600 text-white hover:bg-red-700'
};

export function Button({ variant = 'primary', children, ...props }) {
  return (
    <button 
      className={`px-4 py-2 rounded-lg transition-colors ${buttonVariants[variant]}`}
      {...props}
    >
      {children}
    </button>
  );
}
```

## 3. 다크 모드 구현

### 시스템 설정 기반 다크 모드

```javascript
// tailwind.config.js
module.exports = {
  darkMode: 'class', // 또는 'media'
  // ...
}
```

```javascript
function DarkModeToggle() {
  const [isDark, setIsDark] = useState(false);
  
  useEffect(() => {
    if (isDark) {
      document.documentElement.classList.add('dark');
    } else {
      document.documentElement.classList.remove('dark');
    }
  }, [isDark]);
  
  return (
    <button
      onClick={() => setIsDark(!isDark)}
      className="p-2 rounded-lg bg-gray-200 dark:bg-gray-700 text-gray-900 dark:text-white"
    >
      {isDark ? '☀️' : '🌙'}
    </button>
  );
}
```

## 4. 애니메이션과 트랜지션

### 커스텀 애니메이션

```javascript
// tailwind.config.js
module.exports = {
  theme: {
    extend: {
      animation: {
        'fade-in': 'fadeIn 0.5s ease-in-out',
        'slide-up': 'slideUp 0.3s ease-out',
        'bounce-slow': 'bounce 2s infinite',
      },
      keyframes: {
        fadeIn: {
          '0%': { opacity: '0' },
          '100%': { opacity: '1' },
        },
        slideUp: {
          '0%': { transform: 'translateY(100%)' },
          '100%': { transform: 'translateY(0)' },
        },
      },
    },
  },
}
```

### 호버 효과

## 결론

Tailwind CSS는 강력하고 유연한 도구입니다. 이러한 팁과 트릭을 활용하면 더 효율적이고 일관된 디자인을 만들 수 있습니다.

기억해야 할 핵심 포인트:
- 커스텀 유틸리티 클래스로 재사용성 높이기
- 반응형 디자인으로 모든 기기에서 완벽한 경험 제공
- 애니메이션과 트랜지션으로 생동감 있는 UI 만들기
- 성능 최적화로 빠른 로딩 시간 확보
- 디자인 시스템으로 일관성 있는 브랜드 경험 구축
